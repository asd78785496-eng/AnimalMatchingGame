# 🐾 AnimalMatchingGame (動物配對遊戲) - V1.0
這是一款由我獨立開發的微型「動物配對遊戲」。本專案是我自主研習 **C# 程式語言**與遊戲邏輯設計的實作成果，並透過 **Git / GitHub** 進行完整的版本控制與發布。

## 🎯 專案開發動機與核心亮點

本專案旨在透過實作，將「抽象的邏輯思維」具現化為「可遊玩的遊戲原型」。在開發過程中，我特別專注於以下能力的培養：
*   **物件導向與狀態邏輯**：使用 C# 梳理卡牌點擊、動態配對、重複點擊判定與計分機制的底層邏輯。
*   **資源架構管理**：結構化整理遊戲內的所有資產（Assets），確保專案具備良好的擴充性。

## 🛠️ 開發環境與技術棧

*   **程式語言**：C#
*   **版本控制**：Git / GitHub Desktop。

## 🎮 遊戲核心機制設計

1.  **初始化模組**：遊戲啟動時自動生成並打亂動物卡牌矩陣。
2.  **配對邏輯鏈**：
    *   玩家點擊第一張牌 ➡️ 記錄牌面資訊並翻開。
    *   玩家點擊第二張牌 ➡️ 比對兩者是否一致。
    *   **成功**：鎖定卡牌、增加分數。
    *   **失敗**：觸發延遲機制，自動翻回背面。
3.  **勝負判定**：當所有動物成功配對後，觸發結束結算畫面。

## 📂 專案結構說明

```text

│   └── Textures/         # 動物圖形與 UI 資源
└── RELEASES.md           # 版本發布紀錄
```
